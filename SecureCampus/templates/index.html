Face Recognition, Identifies individuals entering the premises!

The face recognition feature uses the DeepFace library, a deep learning-based facial recognition system, to accurately identify individuals as they enter the premises. This feature enhances security by ensuring that only authorized individuals gain access. It also aids in efficient attendance tracking by automatically logging entries and exits.

Try it now!

Clothes Detection, Automatically detects inappropriate clothing!

The clothes detection feature leverages the YOLO (You Only Look Once) model, a state-of-the-art object detection algorithm, to scan individuals entering the premises. It identifies specific types of clothing, such as shorts or crop tops, that violate the institution's dress code. When inappropriate attire is detected, the system sends an alert to security personnel.

Try it now!

Violence Detection, Detects potential violent behavior!

The violence detection feature employs the YOLO model to continuously monitor surveillance footage for signs of aggressive or violent behavior. The algorithm analyzes body language, movements, and interactions to identify potential conflicts or violent incidents. When such behavior is detected, the system sends alerts to security personnel.

Try it now!
